# Nexus Corporate Landing Page

A premium, modern corporate landing page built with pure HTML, CSS, and JavaScript. Features dark/light mode, animated counters, FAQ accordion, form validation, and smooth micro-interactions.

## 📋 Features

| Feature | Description |
|---------|-------------|
| 🌓 Dark/Light Mode | Toggle themes with localStorage persistence |
| 📱 Fully Responsive | Mobile, tablet, and desktop optimized |
| 🧭 Sticky Header | Navigation bar stays on scroll |
| 📊 Animated Counters | Numbers count up when visible |
| ❓ FAQ Accordion | Expand/collapse questions |
| 📝 Form Validation | Validates name, email, and message |
| 🔔 Toast Notifications | Popup feedback messages |
| 🎨 Glassmorphism UI | Modern blurred card effects |
| 🖱️ Smooth Scrolling | Anchor links scroll smoothly |
| 📈 Scroll Progress Bar | Shows reading progress |
| ✨ Scroll Reveal | Elements fade in on scroll |

---

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Custom styling with CSS variables, Flexbox, Grid
- **JavaScript (ES6+)** - Vanilla JS, Intersection Observer API
- **Font Awesome 6** - Professional icons
- **Google Fonts** - Inter & Space Grotesk

1. Clone the repository:
git clone https://github.com/oltionshumollii/nexus-landing-page.git

2.Navigate to the project folder

3.Open index.html in your browser or use Live Server.
